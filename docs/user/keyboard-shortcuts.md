# Tangentbordsgenvägar

Komplett referens för alla tangentbordsgenvägar i Hitta ansikten.

> **Tips:** Tryck `?` för att visa hjälpöverlägget med de viktigaste genvägarna.

---

## Fil

| Genväg | Funktion |
|--------|----------|
| `Cmd+O` | Öppna fil(er) |
| `Cmd+R` | Ladda om databasen |
| `Cmd+S` | Spara alla ändringar |
| `Escape` | Förkasta ändringar |
| `Cmd+Q` | Avsluta (Windows/Linux) |
| `Cmd+W` | Stäng fönster |

---

## Bildvisning

### Zoom

| Genväg | Funktion |
|--------|----------|
| `+` | Zooma in (håll för kontinuerlig) |
| `-` | Zooma ut (håll för kontinuerlig) |
| `=` | Återställ till 1:1 |
| `0` | Anpassa till fönster |
| `Cmd++` | Zooma in (meny) |
| `Cmd+-` | Zooma ut (meny) |
| `Cmd+=` | Återställ till 1:1 (meny) |
| `Cmd+0` | Anpassa till fönster (meny) |

### Bounding boxes

| Genväg | Funktion |
|--------|----------|
| `Shift+B` | Visa/dölj bounding boxes |
| `b` | Växla alla/enstaka ansikten |

### Visning

| Genväg | Funktion |
|--------|----------|
| `c` | Aktivera auto-centrering på ansikte |
| `C` | Avaktivera auto-centrering |
| `Shift+I` | Visa/dölj "Review Progress" |

---

## Ansiktsgranskning

### Navigation

| Genväg | Funktion |
|--------|----------|
| `↑` / `↓` | Föregående/nästa ansikte |
| `Tab` | Nästa fält |
| `Shift+Tab` | Föregående fält |

### Åtgärder

| Genväg | Funktion |
|--------|----------|
| `A` eller `Enter` | Acceptera föreslagen matchning |
| `Shift+Cmd+A` | Acceptera alla förslag i bilden |
| `I` | Ignorera ansikte |
| `R` | Byt namn / ange namn (rensar fältet) |
| `1-9` | Välj matchningsalternativ |
| `M` | Lägg till manuellt ansikte |
| `X` | Hoppa till nästa fil |
| `Cmd+Z` | Ångra senaste ansiktsåtgärd |
| `Escape` | Avbryt detection / förkasta ändringar |

### Autocomplete

| Genväg | Funktion |
|--------|----------|
| `Tab` | Välj första förslaget |
| `↓` / `↑` | Navigera i förslagslistan |
| `Enter` | Bekräfta valt namn |
| `Escape` | Stäng förslagslistan |

## Plattformsnoteringar

- **macOS**: Använd `Cmd` (⌘)
- **Windows/Linux**: Använd `Ctrl`
- Genvägar med `+` och `-` fungerar med både huvudtangentbordet och numpad
- Håll `+` eller `-` intryckt för kontinuerlig zoom
